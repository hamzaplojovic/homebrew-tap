# Homebrew Tap for Remind

CLI tools by [@hamzaplojovic](https://github.com/hamzaplojovic).

## Installation

```bash
# Add the tap
brew tap hamzaplojovic/remind

# Install remind-cli
brew install remind-cli

# Or directly install without adding the tap
brew install hamzaplojovic/remind/remind-cli
```

## Updating

```bash
brew update
brew upgrade remind-cli
```

## Available Formulas

| Formula | Description | Install |
|---------|-------------|---------|
| remind-cli | AI-powered CLI reminder engine with desktop notifications and Claude Code integration | `brew install hamzaplojovic/remind/remind-cli` |

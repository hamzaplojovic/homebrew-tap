# Homebrew Tap

CLI tools by [@hamzaplojovic](https://github.com/hamzaplojovic).

## Usage

```bash
brew tap hamzaplojovic/tap
```

## Available Formulas

| Formula | Description | Install |
|---------|-------------|---------|
| remind-cli | Smart CLI reminders with AI nudges | `brew install hamzaplojovic/tap/remind-cli` |

## Adding a New Formula

Drop a `.rb` file in `Formula/` and push.
